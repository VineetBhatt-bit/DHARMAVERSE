# Why Here Engine

Milestone 4 turns location explanation into a causal model.

## Engine Shape

Each location now contains `whyHereEngine`:

- `question`: The core question the location must answer.
- `answer`: A concise synthesis of why the place developed where it did.
- `factors`: Selectable causal forces.

## Causal Factor

Each factor contains:

- `id`: Stable identifier.
- `force`: Short name for the causal force.
- `layer`: Primary memory layer.
- `cause`: What physical, cultural, ecological, or human condition created pressure.
- `effect`: What changed because of that condition.
- `evidence`: What users can inspect or experience in the place.
- `connectsTo`: Related memory layers or civilization threads.

## Product Rule

The engine should not say only that a place is important. It should show the chain that made the place possible.
