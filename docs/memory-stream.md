# Memory Stream

Milestone 3 turns static place identity into ordered experience.

## Stream Scene

Each stream scene contains:

- `id`: Stable scene identifier.
- `layer`: Memory layer used for filtering.
- `moment`: Time, phase, or experiential position.
- `title`: Short cinematic title.
- `scene`: Narrative memory text.
- `sensory`: Sound, weather, texture, or atmosphere cue.
- `thread`: Civilization connection surfaced by the scene.

## Interaction

The prototype supports:

- Previous and next scene movement.
- Scene dot indicators.
- Layer filters per selected place.
- Automatic reset when the user switches location.

## Product Rule

Memory Stream scenes should not summarize facts. They should make the place unfold.
